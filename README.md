# Product Calculation Simulator

日本茶の商品購入シミュレーションアプリケーション

## 概要

このアプリケーションは、日本茶販売店向けの商品購入シミュレーターです。お客様が商品を選択して数量を入力し、配送オプションを選択することで、最終的な合計金額を計算できます。

## 機能

### 表示モード
- **商品カタログ**: カテゴリ別に整理された商品リスト（アコーディオン形式）
- **価格計算**: 割引価格の自動計算と表示
- **数量選択**: 各商品の数量選択とリセット機能
- **配送オプション**:
  - 送料選択（無料、¥660、¥880、¥1,000）
  - 代引きオプション（動的料金計算）
  - クール便オプション（+¥330）
- **合計金額表示**: 商品合計、配送料、最終合計の表示
- **納品書印刷**: A4サイズで納品書と控えを印刷（内税表示、会社情報含む）

### 編集モード
- **商品管理**: 商品の追加、編集、削除
- **カテゴリ管理**: ドラッグ&ドロップによるカテゴリ順序変更
- **データ管理**: インポート/エクスポート機能
- **割引設定**: 商品ごとの割引率設定（10%、15%、20%）

## 技術仕様

- **HTML5**: セマンティックマークアップ
- **CSS3**: Flexboxレイアウト、レスポンシブデザイン
- **JavaScript (ES6+)**: DOM操作、ローカルストレージ、イベント処理
- **データ永続化**: LocalStorage API

## 使用方法

### 表示モード（購入シミュレーション）
1. カテゴリをクリックして商品リストを展開
2. 各商品の数量を選択
3. 配送オプションを選択:
   - 送料を選択
   - 代引きの有無を選択（料金は合計金額に応じて自動計算）
   - クール便の有無を選択
4. 最終合計金額を確認

### 編集モード（商品管理）
1. 右上の「編集モードに切替」ボタンをクリック
2. 商品の追加・編集・削除
3. カテゴリ順序の変更（ドラッグ&ドロップ）
4. データのインポート/エクスポート

### 納品書印刷
1. 表示モードで商品と数量を選択
2. 配送オプションを設定
3. 「納品書を印刷」ボタンをクリック
4. 印刷プレビューで内容を確認して印刷

## 商品データ更新方法

商品情報を永続的に保存するには、編集モードでのエクスポート機能を使用して、`script.js`ファイルを更新します。

### ステップ1: 編集モードで商品を管理

1. **編集モードに切り替え**
   - 右上の「編集モードに切替」ボタンをクリック

2. **商品の追加**
   - 商品フォームに以下の情報を入力:
     - 商品名: 商品の名称
     - 価格: 税込価格（¥マークなし、数字のみ）
     - カテゴリ: 既存カテゴリから選択、または新規カテゴリ名を入力
     - 税率: 8%（軽減税率）または10%（標準税率）を選択
     - 割引: 割引対象の場合チェックし、割引率（10%, 15%, 20%）を選択
   - 「保存」ボタンをクリック

3. **商品の編集**
   - 商品一覧から編集したい商品の「編集」ボタンをクリック
   - フォームに現在の情報が表示されるので、必要な箇所を修正
   - 「保存」ボタンをクリック

4. **商品の削除**
   - 商品一覧から削除したい商品の「削除」ボタンをクリック
   - 確認ダイアログで「OK」をクリック

5. **カテゴリ順序の変更**
   - カテゴリ順序管理セクションで、カテゴリをドラッグ&ドロップ
   - 順序が自動保存されます

### ステップ2: データをエクスポート

1. **エクスポート実行**
   - 編集モード内の「カテゴリ順序でエクスポート」ボタンをクリック
   - JavaScriptコードが生成され、テキストエリアに表示されます

2. **コードをコピー**
   - 「クリップボードにコピー」ボタンをクリック
   - または、テキストエリアの内容を手動で選択してコピー（Ctrl+C / Cmd+C）

### ステップ3: script.jsファイルを更新

1. **script.jsファイルを開く**
   - テキストエディタまたはVS Codeで`script.js`ファイルを開きます

2. **defaultProducts部分を検索**
   - ファイル内で `const defaultProducts = [` という行を探します
   - この部分が商品データの配列です

3. **コードを置き換え**
   - `const defaultProducts = [` から対応する `];` までの全体を、エクスポートしたコードで置き換えます
   - 例:
     ```javascript
     // 置き換え前
     const defaultProducts = [
         { id: 1, name: "旧商品1", ... },
         { id: 2, name: "旧商品2", ... }
     ];
     
     // 置き換え後（エクスポートしたコード）
     const defaultProducts = [
         { id: 1, name: "新商品1", ... },
         { id: 2, name: "新商品2", ... },
         { id: 3, name: "新商品3", ... }
     ];
     ```

4. **ファイルを保存**
   - 変更を保存します（Ctrl+S / Cmd+S）

5. **ブラウザで確認**
   - ブラウザでページをリロード（F5 / Cmd+R）
   - 表示モードで更新された商品が反映されていることを確認

### 重要な注意事項

- **税率の設定**: 
  - 食料品（茶類）: 8%（軽減税率対象）
  - お菓子・その他: 10%（標準税率）
  - 税率は納品書の税額計算に使用されます

- **価格の入力**: 
  - すべての価格は**税込価格**で入力してください
  - 納品書では内税計算（税抜価格と税額を自動計算）されます

- **バックアップ**: 
  - `script.js`を更新する前に、必ずバックアップを取っておくことをお勧めします

- **データの永続化**: 
  - エクスポートせずに編集した商品データは、ブラウザのローカルストレージに保存されます
  - しかし、ブラウザのキャッシュをクリアすると失われます
  - 永続的に保存するには、必ず`script.js`を更新してください

## 代引き料金体系

- 合計¥10,000以下: +¥330
- 合計¥10,001～¥30,000: +¥440
- 合計¥30,001以上: +¥660

※代引き料金は、代引き料金自体を含めた最終金額で判定されます
